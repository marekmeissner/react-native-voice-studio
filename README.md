# react-native-voice-studio

React Native module for audio recording

## Installation


```sh
npm install react-native-voice-studio
```


## Usage


```js
import { VoiceStudioView } from "react-native-voice-studio";

// ...

<VoiceStudioView color="tomato" />
```
